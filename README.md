# PSU-DentBook

This repository contains **DentaStream**, an integrated web application for clinical workflow automation and real-time chat built with React, TypeScript, and Vite.

High-level goals:

- Digitize booking/appointment and administrative processes
- Streamline daily clinic operations and reduce scheduling conflicts
- Improve overall patient experience and modernize record keeping

The main frontend app lives in the `dentastream` folder. See `dentastream/README.md` for how to run and develop the app.
