# DentaStream (PSU DentBook Frontend)

DentaStream is a React + TypeScript + Vite web application for PSU DentBook. It focuses on digitizing appointment booking, clinic workflow, and real-time chat between dentists, staff, and patients.

## Features (Current Skeleton)

- Dashboard with placeholders for today&apos;s appointments and queue
- Appointments page with calendar and upcoming visits sections
- Patients page with patient list and record area
- Real-Time Chat page layout for future Socket.IO/WebSocket integration
- Settings page for clinic rules and configuration

## Getting Started

From the `dentastream` folder:

```bash
npm install
npm run dev
```

Then open the shown local URL (default: `http://localhost:5173`).

To build for production:

```bash
npm run build
```
